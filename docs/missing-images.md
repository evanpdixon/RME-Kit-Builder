# Products Using SVG Placeholders (Missing Photos)

These products display SVG line-art icons instead of real product photos.
Add images by setting the `img` property in `assets/js/kit-builder.js` to the WP uploads URL.

Format: `img: S+'YYYY/MM/filename.jpg'` (S = uploads base URL)
